Qwen3-VL Knowledge Distillation for Spatial Reasoning

Multi-stage knowledge distillation pipeline transferring spatial reasoning from a 32B vision-language teacher to a 2B student model. Implements Response SFT, Chain-of-Thought supervision, and Logit KL divergence. Built a 50k+ sample teacher dataset with confidence filtering and spatial reasoning keyword gating. LoRA ablation studies (r=16, r=64) evaluated on MME, MMStar, and CV-Bench with L1/L2/L3 spatial reasoning breakdown.

Knowledge Distillation VLM LoRA CoT Qwen3-VL Python

GitHub
GraspGeom: Egocentric Grasp Affordance Analysis

Computer vision pipeline for extracting grasp affordances from egocentric video to feed robotic manipulation policies. Uses Qwen3-VL-2B to detect the precise frame of first hand-object contact, MoGe2 monocular depth estimation, and SAM3 text-prompted segmentation to produce per-frame 3D point clouds, oriented bounding boxes, principal axes, and candidate contact points — all serialized as structured JSON for downstream robot learning.

Egocentric Video Qwen3-VL SAM Depth Estimation Point Cloud Robot Manipulation

Egocentric Demo Alignment for Robot Learning

Pipeline that converts egocentric human demonstrations into structured robot action trajectories for manipulation policies. Fine-tuned SmolVLA via behavior cloning to map visual observations to end-effector actions. Extracted hand trajectories and interaction events from video, then constructed training datasets through segmentation, temporal synchronization, and smoothing.

Imitation Learning SmolVLA LeRobot Behavior Cloning Egocentric Video

Autonomous Vehicle — Hand Signal Recognition

Real-time hand gesture recognition system for autonomous vehicle navigation on a Jetson Nano. Trained a classification model with TensorFlow and Roboflow on a GPU cluster, then integrated it into a live navigation pipeline using OpenCV with LiDAR-camera sensor fusion for robust scene understanding.
